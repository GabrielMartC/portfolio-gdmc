Libro Interactivo

Desarrollo de una aplicación web interactiva inspirada en los clásicos “Elige tu propia aventura”. El proyecto incluye un sistema de inicio de sesión con vista de administración, donde se gestionan capítulos de la historia, y una interfaz pública que permite al usuario tomar decisiones y explorar distintos finales posibles. El sitio es completamente responsive, combina HTML, CSS y recursos multimedia, e integra formularios y componentes visuales que simulan la experiencia de un producto real.
